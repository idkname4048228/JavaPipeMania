畫面左邊為遊戲關卡的主畫面，右邊為操作介面。

主畫面的遊戲關卡使用讀取檔案的方式製作，而遊戲由於 CardLayout 的排版方式，所以無法新增太多關卡。

操作介面有上一關、下一關、重新開始、向右旋轉 90 度、向左旋轉 90 度等等功能。

基本功能：

建構一個地圖，包含水管、起點、終點(y)

水管: 可上下左右移動或旋轉，依遊戲的玩法決定。(y)

控制：可控制水管移動或旋轉

由畫面上的按鈕控制。(y)

由鍵盤的按鍵控制 。

由滑鼠控制

水管接通時，水會流動。(n)

有計分/計時的機制。(n)

有排行榜的機制，結果存在外部檔案。(n)

地圖，可增加其他裝飾物件，例如，樹木，海等。(n)

有些關卡終點有多個(y)

地圖內的元件，地圖排列的方式，可由外部檔案指定。(y)

提供多關卡，可由外部檔案指定提供一個按鈕，按下後，會有玩法說明的頁面。(n)

遊戲玩法請自訂，請參考相關的遊戲玩法。(y)
